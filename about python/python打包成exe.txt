方法有很多，百度一下就有一堆
这里我只记一个
用pyinstaller进行打包；
下载方式一堆，还是就就记一个
就是它：
pip install pyinstaller
在cmd中输入如上代码。


操作过程：
将cmd的目录切换至（命令：cd 文件路径(注意空格)）需要打包的py文件目录下：

并输入代码，格式为使用命令：  pyinstaller -F 文件名（带后缀py）  ：
[python] view plain copy

 
pyinstaller -F test.py  
常用参数说明：
Cicon=图标路径
-F 打包成一个exe文件
-w 使用窗口，无控制台
-c 使用控制台，无窗口
-D 创建一个目录，里面包含exe以及其他一些依赖性文件
pyinstaller -h 来查看参数
回车后，代码迅速操作，直到操作结束。
